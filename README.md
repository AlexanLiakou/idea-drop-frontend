
# Idea Hub - Front-end

A simple full-stack application to store project ideas. It has a full idea CRUD cycle and a complete AUTH functionality using JWT tokens.

<br>

## Tech Stack

### Front-end

**Main:** [Vite](https://vite.dev/), [React.js](https://react.dev/), [Tanstack Query](https://tanstack.com/query/latest), [Tanstack Router](https://tanstack.com/router/latest),  [Axios](https://www.npmjs.com/package/axios), [Tailwind CSS](https://tailwindcss.com/)

**Packages:** [React Icons](https://www.npmjs.com/package/react-icons), [Lucide React](https://www.npmjs.com/package/lucide-react)


### Back-end

**Main:** [MongoDb](https://www.mongodb.com/), [Mongoose](https://www.npmjs.com/package/mongoose), [Express.js](https://expressjs.com/en/), [Jose JWT Tokens](https://www.npmjs.com/package/jose),

**Packages:** [Bcrypt.js](https://www.npmjs.com/package/bcryptjs), [Cookie Parser](https://www.npmjs.com/package/cookie-parser), [Cors](https://www.npmjs.com/package/cors), [Dotenv](https://www.npmjs.com/package/dotenv)

## Run Locally

### Front-end

Clone the project

```bash
  git clone https://github.com/AlexanLiakou/idea-drop-frontend.git
```

Go to the project directory

```bash
  cd idea-drop-frontend
```

Install dependencies

```bash
  npm install
```

Start the Vite server

```bash
  npm run dev
```

### Back-end

Clone the project

```bash
  git clone https://github.com/AlexanLiakou/idea-drop-backend.git
```

Go to the project directory

```bash
  cd idea-drop-backend
```

Install dependencies

```bash
  npm install
```

Start the Vite server

```bash
  npm run dev
```
